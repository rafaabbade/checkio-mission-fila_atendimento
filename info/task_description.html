Enunciado

Luiza, uma gerente de um centro de atendimento ao cliente, precisa implementar um sistema de fila para gerenciar os atendimentos. Ela quer usar uma fila para organizar a ordem de chegada dos clientes. Crie uma função que simule uma fila de atendimento. A função deve permitir adicionar clientes à fila e atender os clientes na ordem em que chegaram.

Entrada: Uma lista de comandos, onde cada comando é uma string: "chegada ", "atender".

Saída: Uma lista dos clientes atendidos na ordem correta.
